# 丹青遗珍：古画修复大师 (Paintings of the Past)

一款基于现代前端技术重构的高保真拼图游戏。玩家扮演一名古画修复师，通过现代科技手段修复碎裂的古代名画。

## 🎨 游戏设定 (v0.5)
- **主题**：中国传统绘画修复（如《千里江山图》、《清明上河图》）。
- **目标**：在限定时间内将凌乱的古画残片旋转并拼接完整。
- **经济系统**：过关可获得“古钱币”酬报，用于在修复工坊购买辅助设备。

## 🛠️ 修复辅助设备 (道具系统)
1. **【蓝色】绿宝石 (Emerald)**：高频共振设备，使用后自动寻找并拼合两片相邻的碎片。
2. **【金色】立场发生器 (Generator)**：开局使用的强力设备，可以在初始状态下抵消所有碎片的随机旋转（仅对当前关卡生效）。
3. **【紫色】灵笔 (Brush)**：瞬时复位设备，点击使用后，手动将场上所有碎片的旋转角度归零，极大地降低视觉难度。
4. **【红色】时光香 (Incense)**：延长修复时限，为复杂的修复工作争取额外 60 秒时间。

## 🕹️ 核心交互
- **鼠标左键拖拽**：移动拼图碎片。
- **鼠标右键/双击**：旋转拼图。
- **靠近自动吸附**：当两片正确的碎片位置与角度接近时，将自动吸附并合并。

## 🏗️ 架构设计 (全模块化)
项目已完成从单文件到 ES Modules 的彻底重构：
- `js/core/`：核心几何引擎（Delaunay 三角剖分、Voronoi 边界生成、随机采样）。
- `js/game/`：游戏逻辑管理（GameManager、Puzzle 调度器、道具系统、关卡配置）。
- `assets/`：高保真名画资源。
